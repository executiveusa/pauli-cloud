# Claude Code Project Instructions

Pauli Cloud is a portable governance add-on, not an application-specific product.

- Stay on the branch assigned to the Claude Code session.
- Treat large scope as resumable phases.
- Do not stop after planning when safe implementation remains.
- Use `.pauli-cloud/state/ACTIVE_BUILD_STATE.json` as the resume contract.
- Canonical prompts are immutable; create new versions and update hashes.
- Run `npm run check` before each verified checkpoint.
- Never claim a push, PR, browser test, or deployment without evidence.
- Production, irreversible, financial, destructive, and consequential agent actions require explicit approval.
